Promises and Async/Await‚Äã

‚ùì What is the difference between promises and async/await? ‚ú®
üîë Answer: Promises are objects representing eventual completion or failure of an asynchronous operation. Async/Await makes it easier to handle promises by writing cleaner, synchronous-like code.

// Promise Example
fetch("https://api.example.com")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

// Async/Await Example
async function fetchData() {
  try {
    const response = await fetch("https://api.example.com");
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
